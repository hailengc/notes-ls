fixed width: of 800 pixels. Horizontally center. Image right edge will be clipped in a narrow window

auto scale: img width set to 100%, will auto scale together with window resize

fixed height: with 100% width. will end up with wrong scale, can be fixed by given width auto

mix and max on width: image will be clipped if window size smaller than min-width, can be fixed by given a min-width on container or body
